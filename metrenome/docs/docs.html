Introduction

Originally, metronome is a device that counts regular intervals of time. Metronome Coming Soon Page is a responsive template with a useful admin panel, a subscription form and a countdown. Also, it's metro. Perfect combination!

Requirements

The template has been tested in the following browsers: Chrome 26, Firefox 20, Safari 5.1 and 6, Internet Explorer 8 and 9, Android 4 Browser, iPad Safari, Opera Mobile and Chrome for Mobile. It works perfectly in modern browsers, while in older ones it gracefully degrades to work without animation.

Important: Metronome requires Apache 2.2+ web server with PHP 5.2+ module and a configured mail server. If you just open it with a browser on your PC, you will find that admin panel, subscription form and countdown don't work. You can do it to check how the template looks, not how it works.

The subscription form also needs javascript to be turned on in user's browser.

Description

Metronome is a coming soon page which is useful when your site is under construction and you want to spred the word about it.

Subscription Form and Emails

There is a form on the template's main page which allows a visitor to submit his email address. When this happens, the email is saved in the subscription list file (by default it's subscription_list.txt in the root folder). Also, user automatically receives an email which can be customized and turned off in the admin panel.

Another mailing function available is notification emails. These are emails which you can send manually from the admin panel to all your subscribers.

Admin Panel

Admin panel is a tool which allows you to manage template settings and emails. It is available on the following url: http://yoursite/admin.

On the first run you will be asked to set up the password to protect the access to the panel. Any user who opens the admin panel page will have to enter this password. This password can be changed from admin settings page.

Panel's interface is pretty simple: there is a fixed main menu on the left side and a main content area on the right. There are only 4 pages available in the main menu: subscriptions page (which is also the main page) and 3 settings pages.

The main page shows your subscription list and allows you to clear it and to send emails to all subscribers.

Three other pages are admin settings page, email settings page and template settings page. Admin settings are variables for admin panel (password and 2 file paths) which are rarely needed to change. Email settings include all parameters for your subscription and notification emails. Template settings are responsible for the page itself. They are responsible for countdown date and subscription form messages.

Using Metronome

Preparation

Before using this template, you should already have a web hosting and be able to upload files to it. But before doing this, you will need to make a few adjustments to the template. Its files are located in the template folder of the package.

Changing Background

At first, you need to decide what background you want to use on your page. There are 10 images in the backgrounds folder. You can choose one of them or use your own. To change the template's background, replace the original background file (template/img/bg.jpg) with yours.

Content Editing

The second step is to change the heading text, the description text and the social links. It can be done by editing index.html file with your favorite text editor.

<!-- Header -->
<h1 class="header section">Metronome</h1>
<!-- /Header -->

<!-- Description -->
<p class="description section">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Perferendis, non, natus, repellendus in blanditiis explicabo dignissimos amet tenetur quia aperiam culpa cum illum numquam nesciunt voluptatibus itaque nemo beatae inventore!</p>
<!-- /Description -->

<!-- Social links -->
<div class="social-links">
    <a href="#" class="icon-facebook blue"></a>
    <a href="#" class="icon-twitter-alt red"></a>
    <a href="#" class="icon-googleplus yellow"></a>
    <a href="#" class="icon-youtube green"></a>
    <a href="#" class="icon-github-alt red"></a>
</div>
<!-- /Social links -->

Sections of index.html to edit

Coloring

Also, you may want to change some element colors. There are 4 predefined colors: red, blue, green and yellow. They can be applied to the subscription form, the countdown items and the social links. Element's color depends on what html class it has. For example, you want to make the subscription form blue. Then, you should find its class (by default it's red) and change it to blue. The same goes for the rest of colorable elements.

<!-- Subscription form -->
<form class="subscription-form section red" id="subscription-form" novalidate>

<!-- Countdown number -->
<div class="countdown-number green" id="countdown-seconds">

<!-- Social link -->
<a href="#" class="icon-facebook blue"></a>

Colorable elements with highlighted color classes

Uploading Files

All other settings (e.g. countdown date) are changeable via admin panel. Upload all files from the template folder to your web server and open the site in your browser. The page should be completely working including the countdown and the subscription form. After that you will be able to access the admin panel.
